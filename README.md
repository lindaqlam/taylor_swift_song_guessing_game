# Taylor Swift Song Guessing Game 

**Adapted from Colt Steele's RGB Color Game**

Guess the Taylor Swift song given one obscure lyric. That's basically it! 
Now with album modes. Guess with songs from just one album, or her entire discography at once.

(Built with HTML, CSS, and plain ol Vanilla JS!)
